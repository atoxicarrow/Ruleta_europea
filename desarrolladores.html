Acerca de la aplicación y desarrolladores

Sobre la aplicación

Este sitio simula una plataforma de casino centrada en la ruleta europea. Su propósito es académico: practicar maquetación con HTML semántico y estilos con CSS sin frameworks, más el diseño de una navegación coherente.

  • HTML y CSS puros, sin frameworks
  • Diseño para resoluciones de 1728×864 a 1920×1080
  • Contenido ficticio salvo esta página e "Historia y reglas"

Tecnologías utilizadas

HTML5 semántico

Uso de header, nav, main, article, section y footer, además de atributos ARIA.

CSS moderno

Grid, Flexbox, variables CSS, unidades relativas, y control de anchos con max-width.

Accesibilidad

Etiquetas, roles, contraste y navegación por teclado.
